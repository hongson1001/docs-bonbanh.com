# docs-bonbanh.com

Tài liệu kế hoạch & hướng dẫn triển khai cho dự án website mua bán ô tô tham chiếu [Bonbanh.com](https://bonbanh.com/).

## Nội dung trong repo

| File | Mô tả |
|------|--------|
| [DEPLOY.md](./DEPLOY.md) | Hướng dẫn deploy: đẩy docs lên GitHub + deploy production (Angular + NestJS + PostgreSQL trên VPS). |
| [PLAN_OTO_MARKETPLACE_BONBANH_STYLE.md](./PLAN_OTO_MARKETPLACE_BONBANH_STYLE.md) | Kế hoạch chức năng, kiến trúc, giai đoạn, báo giá tham khảo. |

## Stack tham chiếu

Angular (Tailwind + NG-ZORRO) · NestJS · PostgreSQL · object storage (ảnh).
